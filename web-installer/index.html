Install Firmware

  1. Connect your ESP32-S3 to your PC via USB.
  2. Click Install below and select the correct COM port when prompted.
  3. Wait for flashing to complete, then unplug and replug the device.
Web Serial is not supported here. Use Chrome or Edge on desktop. Web Serial requires a secure context (https:// or localhost). Check how this page is being served.

About CC_ISIS

CC_ISIS is a MobiFlight custom firmware device that implements an A320-style ISIS (Integrated Standby Instrument System) on a 4-inch LCD driven by an ESP32-S3 microcontroller.

It displays pitch/bank attitude, airspeed, altitude, barometric pressure setting, and ILS deviation — driven by data from your flight simulator via the MobiFlight Connector.

Source code and wiring instructions: github.com/ccrawford/CC_ISIS
