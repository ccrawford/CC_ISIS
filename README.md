# CC_ISIS - Standalone Standby instrument for the A320

This is a fork of my CC_G5 code (https://github.com/ccrawford/CC_G5) that implements an A320-style standby device as a MobiFlight Community Device

Working Features:
-- Horizon display
-- Speed tape
-- Mach speed display (over 0.45 mach)
-- Altitude tape and rolling numeric display (with NEG indicator)
-- Kohlsman value
-- Slip/Skid indicator ("Ball")


Not done
-- LS Display (CDI diamonds)
-- Speed/Altitude bugs
-- Speed/Altitude bug menu
-- Button handling code
